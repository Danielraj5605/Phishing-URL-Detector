About the Project

This is a personal project developed by Danielraj to bridge the gap between complex cybersecurity threats and user-friendly protection tools.

Phishing remains one of the most common cyber-attacks. This tool provides an accessible way for anyone to verify link safety using modern computational techniques.

Core Features

Real-time Scanning

Instant analysis of URL structures and metadata.

ML Detection

Pattern recognition via trained Logistic Regression models.

Heuristic Analysis

Scans for IP masking, deceptive keywords, and SSL status.
